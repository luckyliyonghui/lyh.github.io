发文流程和相应配置

一.发文步骤：
1.自己新建文件并在文件开头手动加入如下格式，路径为/blog/source/_posts，格式为.md文档

页面配置：

文章配置

2.写完后运行hexo g和hexo d发布
如果不可以，用hexo c && hexo g && hexo d # hexo d 表示执行部署，接hexo s为本地

二.md文章常用格式
标题：以#数代表一二三级别标题
注释：<!--xxx-->，xxx替换
粗体：在文字前后分别加上双星号
斜体：在文字前后分别加上单星号
引用块：在文字前加上右折括号
有序列表：直接用数字加点进行排序
无序列表：在文字前加上短横
代码：在代码前后加上浮空顿号
代码块：在代码前后加上三浮空顿号
分割线：三个短横
链接：>title
图片：>
删除线：文字前后分别加上双波浪号
